# Fetemi Content Studios Dashboard

A modern, premium, and responsive content generation web application built with robust backend integration designed to handle long-running automated content creation tasks smoothly.

## Features

- **Modern Glassmorphism UI**: A sophisticated, green-themed interface that offers a rich, dynamic, and premium user experience.
- **Content Workflow**: Seamless three-stage workflow covering Content Input, Draft Review, and Publishing.
- **N8N Backend Integration**: Uses **n8n** as the backend automation engine. The app features robust polling-based integrations to communicate with n8n webhooks, ensuring smooth handling of long-running content generation tasks.
- **Secure Authentication**: Includes a custom authentication screen requiring username and password to secure the dashboard.
- **Live Feedback**: Provides seamless user feedback and state management across steps during the automated content creation process.

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Vanilla CSS (TailwindCSS avoided in favor of bespoke glassmorphism aesthetics)
- **Backend / Automation**: [n8n](https://n8n.io/) - provides live webhooks and automated workflow processing for content requests.

## Getting Started

### Prerequisites

- Node.js installed on your machine.
- An active n8n instance configured with the appropriate webhooks for content generation.

### Installation

1. Clone the repository
2. Navigate to the project directory:
   ```bash
   cd fetemicontent
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Workflow

1. **Input Phase**: Authenticated users input requirements and parameters for the content they wish to generate.
2. **Generation Phase (n8n)**: A request is sent to the n8n backend via webhooks. The React app actively polls the backend for completion status.
3. **Draft Review**: Once generated by n8n, the draft is presented to the user for review.
4. **Publishing**: Approved drafts are moved to the final publishing stage.
